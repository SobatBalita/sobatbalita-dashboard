# Dashboard Analisis Gizi Anak

Dashboard interaktif berbasis Streamlit untuk analisis data status gizi anak dengan fokus pada:

- Stunting
- Wasting

## Features

- Dataset overview
- Exploratory Data Analysis (EDA)
- Visualisasi data
- Insight dan kesimpulan penelitian

## Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn

## Cara Menjalankan

### Install dependency

```bash
pip install -r requirements.txt
```

### Jalankan streamlit

```bash
streamlit run app.py
```

## Dataset

Dataset digunakan untuk analisis status gizi anak berdasarkan data antropometri:
- umur
- tinggi badan
- berat badan
- jenis kelamin
